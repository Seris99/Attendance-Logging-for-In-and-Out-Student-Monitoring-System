In-and-Out Student Monitoring System
🔹 Project Overview
This project is a prototype subsystem for monitoring student attendance by logging IN and OUT events. It includes:
- Student class → represents student data.
- DatabaseManager → handles persistence of attendance logs.
- AttendanceLogger → manages business logic for logging events.
- GUI (Tkinter) → provides a user interface for logging and viewing attendance records.

🔹 Features
- Add student IN/OUT logs.
- Display logs in a table view.
- Auto-clear input fields after logging.
- Modular design with separation of concerns.
